# 🌟 Overview

**ResearchHive AI** is a futuristic AI-powered multi-agent research assistant system built for the **Microsoft Build AI Hackathon 2026** under the **Agent Swarms** theme.

The platform simulates an intelligent research team where multiple autonomous AI agents collaborate together to perform:

* 🔍 Research & data collection
* 📄 Summarization
* ✅ Fact verification
* 📊 Insight generation
* 🎯 Presentation-ready reporting

Instead of relying on a single AI model, ResearchHive AI uses a **swarm of specialized AI agents** working together in real-time to automate complex research workflows.

---

# 🚀 Problem Statement

Modern research workflows are:

* Time-consuming
* Information-heavy
* Difficult to verify
* Hard to organize
* Inefficient for rapid decision making

Researchers, students, startups, and professionals spend hours collecting and validating information manually.

---

# 💡 Our Solution

ResearchHive AI introduces a collaborative AI swarm ecosystem where multiple intelligent agents coordinate together like a professional research team.

Each agent specializes in a specific task and communicates with other agents to generate:

✅ Accurate research
✅ Verified insights
✅ Concise summaries
✅ Analytical findings
✅ Presentation-ready reports

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/research-hive-ai.git
```

---

## 2️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000

MONGODB_URI=your_mongodb_uri

OPENAI_API_KEY=your_openai_key

AZURE_OPENAI_ENDPOINT=your_azure_endpoint

AZURE_OPENAI_API_KEY=your_azure_key
```

---

# ▶️ Run Frontend

```bash
npm run dev
```

---

# ▶️ Run Backend

```bash
npm run start
```
